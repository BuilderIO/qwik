# Static Assets

使用 `/public` 文件夹存储所有静态资源，例如图像、字体等。

所有静态资源都从您服务器的根目录（例如`/`之于` https:example.com/`）文件夹中提供，并叠加在您的路由下（路由优先）。

```
public/
├── favicon.ico           # https://example.com/favicon.ico
└── images/
    └── logo.png          # https://example.com/images/logo.png
```
