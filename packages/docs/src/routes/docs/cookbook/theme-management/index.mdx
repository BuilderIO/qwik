# Theme management

In modern website we have seen that it has the feature of dark and light theme that creates a good impact on the users. They can change the theme according to their prefrences.

To achive this we are also using tailwindcss for styling.

First we have to install tailwindcss in our website. To add tailwind in Qwik project is very simple. You have to just run this command in terminal.

```cmd
npm run qwik add tailwind
```

You have to enable the darkmode from your ```tailwind.config.js``` file. It shoul look like this.

```js
module.exports = {
  content: ["./src/**/*.{js,ts,jsx,tsx,mdx}"],
  darkMode: "media",
  theme: {},
  plugins: [],
};
```

> NOTE:
>
> - If you want to fully rely on user preference at the OS-level, you're all set and ready to go. You do not need to do anything else.
> - By default, this uses the prefers-color-scheme CSS media feature and you can start applying your `dark:{class}` classes.




## Toggling dark mode manually


If you want to support toggling dark mode manually instead of relying on the operating system preference, use the class strategy instead of the media strategy:

```js
module.exports = {
  content: ["./src/**/*.{js,ts,jsx,tsx,mdx}"],
  darkMode: "class",
  theme: {},
  plugins: [],
};

```

Using `darkMode: "class"`, the dark classes will be applied whenever the `dark` class is present earlier in the HTML tree, instead of relying on prefers-color-scheme. This means that we can use this opportunity to support both the user’s system preference and a manually selected mode.

> NOTE:
>
> - This is useful if you want your users to choose a different theme than the one they are using on their OS.



Create a new `ThemeScript` component that will contains our theme management logic:

- User preference for our Qwik application theme takes precedence over the OS-theme, using `localStorage`;
- If the user preferece isn't set in `localStorage`, use the OS-theme;
- Attach an event listener to OS-theme's change;
- Use 'light' theme by default.

```tsx
import { component$, sync$, useOnDocument, $ } from '@builder.io/qwik';

export const colorSchemeChangeListener = (
  onColorSchemeChange: (isDark: boolean) => void
) => {
  const listener = ({ matches: isDark }: MediaQueryListEvent) => {
    onColorSchemeChange(isDark);
  };

  window
    .matchMedia('(prefers-color-scheme: dark)')
    .addEventListener('change', (event) => listener(event));

  return () =>
    window
      .matchMedia('(prefers-color-scheme: dark)')
      .removeEventListener('change', listener);
};

export const ThemeScript = component$(() => {
  const UserColorThemeKey = 'user-preference-color-theme';

  useOnDocument(
    'DOMContentLoaded',
    sync$(() => {
      const userPreference = localStorage.getItem(UserColorThemeKey);

      if (userPreference) {
        if (userPreference === 'dark') {
          document.documentElement.classList.add('dark');
        } else {
          document.documentElement.classList.remove('dark');
        }
      } else if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
        document.documentElement.classList.add('dark');
      } else {
        document.documentElement.classList.remove('dark');
      }
    })
  );

  useOnDocument(
    'DOMContentLoaded',
    $(() => {
      return colorSchemeChangeListener((isDark) => {
        if (!(UserColorThemeKey in localStorage)) {
          if (isDark) {
            document.documentElement.classList.add('dark');
          } else {
            document.documentElement.classList.remove('dark');
          }
        }
      });
    })
  );

  return null;
});
```

Add this component to your `router-head.tsx` component:
```tsx
export const RouterHead = component$(() => {
  return (
    <>
      <title>{head.title}</title>

      // ... all your other links, tags, and scripts

      <ThemeScript />
    </>
  );
});
```

Then in any component you can add your theme toggle code, to let your users manually change the theme.

```tsx
import { component$, event$, useStyles$ } from '@builder.io/qwik';
import { SunAndMoon } from './sun-and-moon';
import themeToggle from './theme-toggle.css?inline';

export const ThemeToggle = component$(() => {
  useStyles$(themeToggle);

  const UserColorThemeKey = 'user-preference-color-theme';

  const onClick$ = event$(() => {
    const currentTheme =
      localStorage.getItem(UserColorThemeKey) ??
      (window.matchMedia('(prefers-color-scheme: dark)').matches
        ? 'dark'
        : 'light');

    const nextTheme = currentTheme === 'dark' ? 'light' : 'dark';
    localStorage.setItem(UserColorThemeKey, nextTheme);

    if (nextTheme === 'dark') {
      document.documentElement.classList.add('dark');
    } else {
      document.documentElement.classList.remove('dark');
    }
  });

  return (
    <button
      type="button"
      class="theme-toggle"
      id="theme-toggle"
      title="Toggles light & dark"
      aria-label={'Toggles light & dark'}
      aria-live="polite"
      onClick$={onClick$}
    >
      <SunAndMoon />
    </button>
  );
});

```
